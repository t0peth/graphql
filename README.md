# GraphQL Server

GraphQL Server is a simple spring boot project for read data from a Postgre database

### Techstack
- Java: 17
- Maven: 3.9.8
- Spring boot: 3.3.2

## Installation

Use [Maven](https://maven.apache.org/download.cgi) to install the project.

```maven
mvn clean install
```
